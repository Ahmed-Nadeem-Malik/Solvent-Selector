# Solvent-Selector
A simple web app that selects solvents based on the users required reaction type using a small dataset. Built using Python as Flask as the front end using Jinja2 templates to display teh HTML and CSS. For th ebackend just regular Python is used along with RDkit to display the molecular structure of the solvent.

How to run: Good idea to run this locally, pip install Flask and RDkit and just run the python file "app.py"
